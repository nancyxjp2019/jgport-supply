## 2.2 M7实施约束
- 首版报表统一指标版本为 `v1`，接口支持显式传入 `metric_version`；若传入未知版本，服务端必须阻断。
- 管理后台仪表盘与业务看板仅允许 `operations/finance/admin + operator_company + admin_web` 访问。
- 小程序轻量报表首版仅允许 `operations/finance/admin + operator_company + miniprogram` 访问，不向客户/供应商/仓库开放经营金额汇总。
- 合同执行率只统计已进入执行阶段的合同：`生效中`、`数量履约完成`、`已关闭`、`手工关闭`、`已归档`。
- 报表“当日”统一按 `Asia/Shanghai` 自然日计算。
- 库存周转按含今日的最近30个自然日计算，分母小于等于0时返回0。
- 报表快照只追加，不覆盖历史记录。
- 本轮不实现：
  - `/reports/admin/multi-dim`
  - 每日闭环扫描任务与履约滞留扫描任务
  - 事件触发增量刷新编排
  - 报表导出、筛选器、钻取与口径重算
