## 2.2 M8-01实施约束
- 管理后台页面仅面向运营侧角色，不给客户/供应商/仓库开放入口。
- 所有用户可见信息必须中文显示。
- 页面视觉必须遵循 `docs/V6现行高保真原型说明.md`，不得自行改写指标名称和状态文案。
- `demo` 模式仅用于前端预览与评审，不得作为生产运行模式。
- 代理联调模式下，由 Vite 开发代理在服务端注入身份头；浏览器端不得暴露后端代理密钥。
- 本轮不实现：
  - 小程序代码
  - 登录、Token、刷新机制
  - 合同/订单/资金/库存页面
  - 多维管理报表页面
