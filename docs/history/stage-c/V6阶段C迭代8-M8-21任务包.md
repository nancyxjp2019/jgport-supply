## 2. 迭代8第二十一批范围（M8-21 多维报表与导出首批）
- 目标：落地管理后台多维报表与导出首批能力，补齐 `M7` 遗留的后台多维查询与最小导出闭环。
- 模块边界：本轮处理 `admin-web + backend报表接口补齐 + M7既有口径复用 + docs`；不新增多维钻取、不新增口径重算任务编排、不新增批量导出调度。
- 首批实现边界：
  - 后端开放 `GET /reports/admin/multi-dim`，支持按合同方向、单据状态、退款状态分组与筛选。
  - 后端开放 `GET /reports/admin/multi-dim/export`，支持当前筛选条件的 CSV 导出。
  - 管理后台新增 `/reports-multi-dim` 页面与导航入口。
  - 页面支持分组方式、合同方向、单据状态、退款状态、日期区间筛选。
  - 页面支持汇总卡片回看与单击导出 CSV。
  - 演示模式补齐多维报表 mock 与单测。
- 本轮不实现：
  - 多维钻取、联动明细跳转、图形钻取
  - 报表口径重算任务编排
  - 批量导出任务队列与异步通知
