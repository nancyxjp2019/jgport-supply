- 首批实现边界：
  - 合同列表支持按方向、状态筛选，并区分采购/销售合同。
  - 合同详情支持回看阈值快照、审批意见、油品明细、下游待处理任务与图谱摘要。

  - 合同手工关闭动作继续复用现有 `合同关闭差异台`，不在本轮重复建设。
- 本轮不实现：
  - 审计日志中心独立页面
  - 参数与字典配置页
  - 单据链路追踪独立页面
  - 合同批量审批、批量关闭、批量导入
