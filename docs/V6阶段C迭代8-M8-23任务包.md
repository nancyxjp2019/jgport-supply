## 2. 迭代8第二十三批范围（M8-23 会话续期与后端细粒度鉴权首批）
- 目标：补齐真实会话续期能力，并把关键动作接口的后端权限与页面按钮边界对齐。
- 模块边界：本轮处理 `backend鉴权 + miniprogram会话续期 + docs`；不改权限配置后台、不做组织级动态权限策略。
- 首批实现边界：
  - 新增会话续期接口：`POST /access/session/refresh`。
  - 小程序在 Bearer 令牌过期后自动触发续期，续期成功自动重试一次原请求。
  - 多维报表导出接口后端权限收口为 `finance/admin`，运营仅可查询报表。
  - 资金列表与详情后端收口为“运营可读、财务/管理员可写”，与页面只读语义对齐。
  - 续期动作写入审计，权限失效或归属不匹配时拒绝续期。
- 本轮不实现：
  - 多端统一 SSO 与跨端单点注销
  - 权限变更审批流与策略中心
  - 导出异步任务中心
