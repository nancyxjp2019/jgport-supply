## 2. 迭代8第四批范围（M8-04 小程序本地登录与会话守卫）
- 目标：补齐小程序本地登录页、角色切换与页面守卫，支撑微信开发者工具中的本地联调与页面评审。
- 模块边界：本轮只处理 `miniprogram/` 的本地登录、会话同步、报表页守卫与文档；不进入真实微信登录、后端透传换会话、订单/待办/执行页面。
- 首批实现边界：
  - 新增 `MINI-LOGIN-01` 本地登录页。
  - 小程序支持按冻结角色进行本地身份切换。
  - `MINI-REPORT-01` 未登录时强制回到登录页。
  - 报表页增加当前身份展示、切换角色与退出动作。
  - 新增本地会话工具与 Node 级回归测试。
- 本轮不实现：
  - 微信 `wx.login`
  - 服务端会话换取与续期
  - 小程序真实身份透传
  - 订单/待办/执行回执页面

## 2.2 M8-04 实施约束
- 所有用户可见文案必须中文显示。
- 小程序本地登录只用于开发者工具联调与页面评审，不得误表述为真实微信登录。
- 未登录不得默认赋予运营身份，必须显式选择角色后才能进入报表页。
- 运营、财务、管理员可见经营快报；客户、供应商、仓库仅用于验证中文阻断提示。
- 本轮不引入第三方小程序 UI 库，继续沿用原生小程序执行链。
