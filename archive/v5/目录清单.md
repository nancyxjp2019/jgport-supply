# 新项目基座目录清单

## 已包含
- `docs/`：业务、接口、状态机、数据库、安全、部署、对齐检查与重建说明。
- `backend/`：V5 业务后端代码、Alembic 迁移、核心初始化脚本、PostgreSQL 18 配置样例。
- `admin-web/`：Vue3 + Element Plus + Vben Admin 初始化占位目录与环境变量模板。
- `miniprogram/`：小程序执行链代码基座。

## 已排除
- `tests/uat/` 及其他 UAT 自动化脚本。
- 后端 UAT 路由与 UAT schema。
- 小程序 `utils/uat_runner.js`。
- 旧后台静态管理台 `backend/app/web/admin_console`。

## 备注
- `backend/app/main.py` 已移除后台静态页挂载与 `/admin-ui` 重定向，默认只承载 API。
- `backend` 默认数据库已切换为 PostgreSQL 18 连接串示例。
- 新项目开发环境统一使用 Conda 环境 `jgport`。
